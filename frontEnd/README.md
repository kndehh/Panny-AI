# Panny — AI Companion

This is a Vite + React + TypeScript project scaffold for Panny — a calming AI companion.

Run locally:

```powershell
npm ci;
npm run dev;
```

Features: 
- React + TS
- Tailwind CSS
- Zustand state management + localStorage persistence
- Framer Motion for smooth transitions
- React Router for navigation
- React Three Fiber for a subtle 3D background
- Soft design language and glassmorphism

This project is a starting point that implements the UI, 3D background wrapper, Zustand store and basic chat UI. Next steps: add semantic UI components from shadcn and complete the 3D scene and audio behavior for accessibility and performance.
